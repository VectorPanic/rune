(static) addEventListener(target, type, listener, optionsopt, useCaptureopt, scopeopt) → {function}

The EventTarget method addEventListener() sets up a function that will be called whenever the specified event is delivered to the target. Common targets are Element, Document, and Window, but the target may be any object that supports events (such as XMLHttpRequest).
Parameters:
Name Type Attributes Description
target Object Object that applies the current event listener.
type string Case-sensitive string representing the event type to listen for (Use static class constants for this).
listener function The object which receives a notification (an object that implements the Event interface) when an event of the specified type occurs. This must be an object implementing the EventListener interface, or a JavaScript function.
options Object <optional>
 An options object that specifies characteristics about the event listener.
useCapture boolean <optional>
 Boolean indicating whether events of this type will be dispatched to the registered listener before being dispatched to any EventTarget beneath it in the DOM tree.
scope Object <optional>
 The scope that the listener executes within.
